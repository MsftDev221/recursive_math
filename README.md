# How to use?

simply run the .jar file by locating it in a terminal, and then typing

```
java -jar recursive_math.jar
```

### You'll have to use a single whitespace between each element.

- This is correct: 
```
"1 + 2 * 3 / 2 + 1 * 54"
```

- These are incorrect: 
```
"1+2*3/2+1*54"
" 1+    2*3 /  2+     1 *54"
```
